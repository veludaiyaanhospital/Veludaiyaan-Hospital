# Main Website (Static)

This folder contains the public hospital website pages.

## Files

- `index.html`
- `about.html`
- `services.html`
- `doctors.html`
- `facilities.html`
- `contact.html`
- `site.js`
- `hero.jpg`
- `LOGO.png`
- `logo1.jpg`

## Open locally

Open `index.html` directly in browser.

## Patient Gateway button

Footer "Patient Gateway" button is wired through `site.js`:
- local file mode -> opens `http://localhost:3000/patient/login`
- hosted mode -> opens `/patient/login`
